# PlantCare Tracker (Multipage Web App)

Natur-inspiriertes, responsives Frontend für einen Web-Pflanzen-Tracker mit mehreren Seiten.

## Seiten

- `index.html` – Dashboard mit Pflanzenkarten, Status und Mini-Fotogalerien
- `add-plant.html` – Formular zum Hinzufügen neuer Pflanzen
- `diagnose.html` – Kamera/Upload-Bereich und Diagnoseergebnis mit Empfehlungen
- `history.html` – Pflegechronologie und einfache Verlaufsvisualisierung
- `settings.html` – Benachrichtigungs- und App-Einstellungen
- `community.html` – Community-Feed mit Beiträgen und Kommentaren

## Design

Die Anwendung nutzt eine natürliche Farbpalette gemäß Vorgabe:

- Dunkelgrün `#1B4D3E`
- Beige `#F1E0C6`
- Hellgrün `#A8D08D`
- Braun `#8B4513`
- Weiß `#FFFFFF`
- Sanftes Gelb `#F6E58D`
- Hellgrau `#D3D3D3`

Typografie:

- Überschriften: **Poppins**
- Fließtext: **Open Sans**

## UX-Highlights

- Aktive Navigation pro Seite
- Hamburger-Menü in der mobilen Ansicht
- Hover-Effekte auf Karten und Buttons
- Sanfter Fade-Up-Seitenübergang
- Toast-Benachrichtigung am unteren Bildschirmrand
- Responsives Layout:
  - Mobil: einspaltig
  - Tablet/Desktop: mehrspaltige Darstellung je nach Seite

## Nutzung

Die App ist statisch aufgebaut. Zum lokalen Start genügt es, `index.html` im Browser zu öffnen.
